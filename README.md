# UK Housing Market Analysis

## Overview
This project analyses housing price data to identify patterns and key factors influencing property values.

## Dataset
- Source: Public dataset (Boston Housing)
- Contains information on housing prices and features such as crime rate, number of rooms, and property tax.

## Tools Used
- Python (Pandas, Matplotlib)

## Analysis Performed
- Data loading and exploration
- Price distribution analysis
- Correlation analysis between features and price

## Key Insights
- House prices show a wide distribution
- Certain features strongly correlate with price (e.g., number of rooms)
- Data can support predictive modelling

## Skills Demonstrated
- Data cleaning
- Exploratory Data Analysis (EDA)
- Data visualisation
- Analytical thinking
